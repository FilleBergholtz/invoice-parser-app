<task type="auto">
  <name>Create resources.qrc and compile with pyside6-rcc</name>
  <files>src/ui/resources.qrc, src/ui/resources_rc.py</files>
  <action>
Create src/ui/resources.qrc in Qt resource XML format: one qresource block with a prefix (e.g. / or /icons), file elements for each asset (e.g. assets/icons/app.svg, assets/icons/open.svg). Paths relative to .qrc file. Run pyside6-rcc on it to produce src/ui/resources_rc.py. Ensure resources_rc is importable (e.g. from src.ui import resources_rc).
  </action>
  <verify>python -c "import src.ui.resources_rc; print('resources_rc ok')" and QIcon(':/icons/app') or similar loads.</verify>
</task>
